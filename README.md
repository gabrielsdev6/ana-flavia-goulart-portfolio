# Ana Flávia Goulart — Beauty Artist

Site institucional desenvolvido para **Ana Flávia Goulart**, beauty artist com atelier em Piumhi, MG.

**[→ Ver site ao vivo](https://ana-flavia-goulart.vercel.app)**

---

## Sobre o projeto

Site de portfólio profissional com foco em conversão — apresenta os serviços, exibe os trabalhos realizados e direciona o agendamento direto pelo WhatsApp.

## Funcionalidades

- Galeria de trabalhos com lightbox e navegação por ângulos
- Formulário de agendamento integrado ao WhatsApp
- Cursor customizado e animações de entrada
- Design responsivo para mobile e desktop
- SEO otimizado com Open Graph e structured data (JSON-LD)
- Vercel Analytics integrado

## Stack

- HTML5
- CSS3
- JavaScript vanilla

## Rodando localmente

Sem dependências. Basta abrir o `index.html` no navegador ou usar o Live Server do VS Code.

## Deploy

Hospedado na [Vercel](https://vercel.com).
